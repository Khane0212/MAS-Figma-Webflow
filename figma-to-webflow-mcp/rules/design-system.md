# Design System Rules

## Màu sắc (Colors)

### Nguyên tắc đặt tên biến màu
```
--color-[purpose]-[variant]-[state]
```

### Ví dụ
- `--color-primary-default` - Màu chính mặc định
- `--color-primary-hover` - Màu chính khi hover
- `--color-secondary-default` - Màu phụ mặc định
- `--color-background-light` - Background sáng
- `--color-text-primary` - Text chính
- `--color-text-secondary` - Text phụ
- `--color-border-default` - Border mặc định

## Typography

### Nguyên tắc đặt tên
```
--font-[property]-[variant]
```

### Ví dụ
- `--font-family-heading` - Font cho heading
- `--font-family-body` - Font cho body text
- `--font-size-h1` - Size cho H1
- `--font-size-body` - Size cho body
- `--font-weight-bold` - Weight bold
- `--line-height-tight` - Line height chặt
- `--letter-spacing-wide` - Letter spacing rộng

## Spacing & Layout

### Scale chuẩn (REM)
- `--space-xs`: 0.25rem (4px)
- `--space-sm`: 0.5rem (8px)
- `--space-md`: 1rem (16px)
- `--space-lg`: 1.5rem (24px)
- `--space-xl`: 2rem (32px)
- `--space-2xl`: 3rem (48px)
- `--space-3xl`: 4rem (64px)
- `--space-4xl`: 6rem (96px)

### Container
- `--container-max`: 80rem (1280px)
- `--container-padding`: 1.5rem (24px)

## Breakpoints
- Mobile: < 768px
- Tablet: 768px - 991px
- Desktop: > 991px
- Large: > 1280px
