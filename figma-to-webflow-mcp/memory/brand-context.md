# Brand Context - Ngữ cảnh ngành nghề đã nhận diện

## Template cấu trúc

```markdown
## [Tên dự án/Brand]

### Ngành nghề (Industry)
- Primary: [Ngành chính]
- Secondary: [Ngành phụ - nếu có]

### Đối tượng khách hàng (Target Audience)
- Demographics: [Tuổi, giới tính, vị trí địa lý]
- Psychographics: [Sở thích, giá trị, hành vi]
- Pain points: [Vấn đề cần giải quyết]

### Tính cách thương hiệu (Brand Personality)
- Tone: [Professional/Friendly/Luxury/Playful/Techy]
- Values: [Giá trị cốt lõi]
- Voice: [Cách thức giao tiếp]

### Từ khóa ngữ nghĩa (Semantic Keywords)
- Primary: [Từ khóa chính liên quan đến ngành]
- Secondary: [Từ khóa phụ]
- Action: [Động từ hành động]

### Tham khảo đối thủ (Competitor References)
- [Đối thủ 1] - [Điểm mạnh/yếu]
- [Đối thủ 2] - [Điểm mạnh/yếu]

### Đặc thù UI/UX theo ngành
- Patterns phổ biến: [Các pattern UI đặc trưng]
- Trust signals: [Yếu tố tạo niềm tin]
- Conversion elements: [Yếu tố chuyển đổi]
```

## Ví dụ: Healthcare Platform

```markdown
## MediCare Plus

### Ngành nghề
- Primary: Healthcare / Telemedicine
- Secondary: Health Tech

### Đối tượng khách hàng
- Demographics: 25-65 tuổi, cả giới, đô thị
- Psychographics: Quan tâm sức khỏe, bận rộn, tech-savvy
- Pain points: Khó đặt lịch, thiếu thông tin bác sĩ, chờ đợi lâu

### Tính cách thương hiệu
- Tone: Professional, Trustworthy, Caring
- Values: Accessibility, Transparency, Quality care
- Voice: Clear, empathetic, authoritative

### Từ khóa ngữ nghĩa
- Primary: patient, doctor, appointment, clinic, health, medical, care
- Secondary: portal, record, prescription, specialist, telehealth
- Action: book, consult, diagnose, treat, monitor

### Tham khảo đối thủ
- Teladoc - Market leader, UI hơi dated
- Amwell - Good UX, kém về mobile
- MDLive - Fast booking, limited specialties

### Đặc thù UI/UX
- Patterns: Booking flows, doctor profiles, symptom checkers
- Trust signals: Certifications, reviews, statistics
- Conversion: Easy booking, clear CTAs, insurance info
```
