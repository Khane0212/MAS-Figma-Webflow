⚠ SAI LỆCH SO VỚI MAPPING PLAN

So sánh DOM template gốc (Finsweet) vs mapping_plan_Privacy Policy_Privacy Policy.md của dự án 

[DIFF-1] Tên section:
  Template gốc : "section_privacy"
  Mapping Plan  : "section_privacy-policy"
  → Cần đổi tên section trên Webflow

[DIFF-2] Cấu trúc content:
  Template gốc : Dùng div.text-rich-text.w-richtext (1 Rich Text block)
  Mapping Plan  : Từng element riêng: privacy-policy_header, privacy-policy_content, privacy-policy_list-group, privacy-policy_contact-info
  → Cần rebuild cấu trúc theo mapping plan

[DIFF-3] Wrapper header:
  Template gốc : max-width-small align-center (không có wrapper header riêng)
  Mapping Plan  : privacy-policy_header (flex-column, gap: 1rem)
  → Cần thêm wrapper riêng

[DIFF-4] Contact info block:
  Template gốc : Không có section contact-info trong main content
  Mapping Plan  : Có privacy-policy_contact-info với email + phone
  → Cần build thêm block này

[DIFF-5] List items:
  Template gốc : ul > li (trong rich text)
  Mapping Plan  : privacy-policy_list-group > privacy-policy_list-item > list-bullet + p
  → Cần rebuild list với custom structure

[MATCH] Padding structure:
  Ca hai dung: padding-global > container-large

[MATCH] Heading:
  Ca hai dung: h1 heading-style-h1 text-align-center voi text "Privacy Policy"
